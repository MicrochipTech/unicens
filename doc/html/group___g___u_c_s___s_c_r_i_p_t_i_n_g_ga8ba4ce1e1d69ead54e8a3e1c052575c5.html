Ucs_Return_t Ucs_Ns_SynchronizeNode ( Ucs_Inst_t *  self,
uint16_t  node_address,
uint16_t  node_pos_addr,
Ucs_Rm_Node_t *  node_ptr,
Ucs_Ns_SynchronizeNodeCb_t  result_fptr 
)

Synchronizes a node which is required to use RoutingManagement, Scripting and GPIO/I2C functionality.

Parameters
selfThe UNICENS instance.
node_addressThe node address.
node_pos_addrThe node position address.
node_ptrReference to a node structure which is referenced within the routing structures.
result_fptrMandatory result callback function.
Returns
Possible return values are shown in the table below.
Value Description
UCS_RET_SUCCESS No error
UCS_RET_ERR_NOT_AVAILABLE This function is not available if the manager is enabled.
UCS_RET_ERR_PARAM At least one parameter is NULL.
UCS_RET_ERR_NOT_INITIALIZED UNICENS is not initialized
UCS_RET_ERR_BUFFER_OVERFLOW UNICENS cannot create an internal administration object.
Check if the configuration value of UCS_NUM_REMOTE_DEVICES is sufficient.
UCS_RET_ERR_API_LOCKED The API is locked.
